# GitHub PAT Anleitung

Der GitHub Personal Access Token wird nur für das initiale Bootstrap benötigt.

Empfohlene Mindestberechtigungen:

- Repository erstellen oder Zielrepository schreiben
- Contents: Read/Write
- Actions Secrets: Read/Write
- Actions Variables: Read/Write
- Workflow: Read/Write

Nach Abschluss des Bootstrap sollte der PAT widerrufen oder ablaufen gelassen werden.
