Entra App Bootstrap

No-install Flow: App Registrations, Admin Consent und browserbasierter GitHub Secret Import.

1. Daten erfassen

Nächster Schritt: Diese Seite wird nach dem Enduser-Bootstrap automatisch im Zielrepository geöffnet und übernimmt Tenant-/Repository-Daten aus der URL.

2. Invoke-Befehl

No-install: Der Befehl installiert keine PowerShell-Module und keine NuGet-Pakete. Die GitHub-Secrets werden danach im Browser aus der Datei github-secrets.json importiert.
Bitte zuerst erzeugen.

3. GitHub Secrets importieren und finalen Check starten

Nach dem PowerShell-Lauf und dem Admin Consent hier die lokal erzeugte Datei .out\entra-bootstrap\github-secrets.json auswählen. Der GitHub Token wird nur im Browser-Arbeitsspeicher genutzt und nicht gespeichert.
